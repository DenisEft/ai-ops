# Memory state check
- Прочитай `memory/state.json` → сравни `lastDailyWritten` с текущей датой
- Если не совпадает → допиши в daily-файл (или создай), обнови state, закоммить
- Если совпадает → просто проверь, что daily-файл существует и не пустой
- Старые daily-файлы > 7 дней → пересмотри на предмет переноса в MEMORY.md
