# Умный дом — План интеграции (07.05.2026)

## Текущее состояние

**Система:** MajordomoSL (SmartLiving) на 192.168.0.111
**Устройства:** 6 реле (свет) в 4 комнатах
**Проблемы:** Connect-сервер не подключен, модуль systeminfo не найден

---

## 🟢 Этап 1: Управление через Telegram (быстрый старт)

### Команды для Лоры:

**Базовые команды света:**
- «включи свет» / «выключи свет» (в текущей комнате или все)
- «включи свет на кухне» / «в коридоре» / «в гостиной» / «у Дианы»
- «какой свет горит?» — показать статус всех устройств

**Массовые команды:**
- «включи весь свет» — все 6 реле ON
- «выключи весь свет» — все 6 реле OFF
- «выключи свет в комнатах» — оставить только коридор

**Медиа:**
- «включи музыку» / «пауза» / «стоп»
- «громче» / «тише»

### Техническая реализация:
```
POST /objects/?method=Relay01.SWITCH (кухня)
POST /objects/?method=Relay02.SWITCH (детская)
POST /objects/?method=Relay03.SWITCH (гостиная)
POST /objects/?method=Relay04.SWITCH (коридор)
POST /objects/?method=Relay06.SWITCH (подсветка кухни)
```

## 🟡 Этап 2: Автоматизация и сценарии

### Сценарий «Доброе утро» (7:00-8:00)
- Включить свет в коридоре
- Включить свет на кухне
- Включить подсветку кухни
- *(опционально) включить музыку*

### Сценарий «Спокойной ночи» (23:00-00:00)
- Выключить весь свет кроме коридора
- *(или) выключить всё полностью*

### Сценарий «Кино / Вечер»
- Выключить яркий свет
- Оставить подсветку кухни
- Убавить громкость

### Расписание через cron:
- 7:00 — Доброе утро (будни)
- 23:00 — Спокойной ночи
- 12:00 — Проверка: «не забыл ли выключить свет?»

## 🟠 Этап 3: Умные уведомления

### Что отслеживать:
- Свет горит более 3 часов без изменений → напомнить
- Все устройства включены после 23:00 → уведомить
- Свет в детской горит, но Денис на работе → проверить

### Как реализовать:
