## API

### Подтверждённые методы управления

```
POST http://192.168.0.111/objects/?method=Relay03.SWITCH
# (с cookies сессии)
# Переключает реле вкл/выкл
```

### Другие эндпоинты

- `/ajax/devices.html?op=clicked&object=Relay03` — подтверждение клика (возвращает `[]`)
- `/admin.php?action=devices` — страница устройств (HTML)
- `/objects/?method=RelayXX.SWITCH` — переключение реле

### Что НЕ работает напрямую

- `/index.php?_module=devices&...` — возвращает полный HTML вместо JSON
- WebSocket порт 8001 — недоступен через HTTP (только ws:// протокол)
