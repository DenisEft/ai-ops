# 2026-05-09 16:30 - Пользовательские метрики

## Реализация системы произвольных системных метрик

### Бэкенд
- **`backend/src/services/metrics-config.js`** — новая система управления метриками
  - `loadMetricsConfig()` — загрузка конфигурации из `metrics-config.json`
  - `addCustomMetric()` — добавление пользовательской метрики
  - `removeMetric()` — удаление метрики
  - `toggleMetric()` — включение/выключение
  - `updateMetric()` — обновление конфигурации
  - `collectMetricData()` — сбор данных с поддержкой методов:
    - `command` — произвольная CLI команда с опциональным regex
    - `nvidia-smi` — данные GPU
    - `systeminfo` — systeminformation
    - `prometheus` — llama-server metrics
    - `thermal` — температура sysfs
    - `filesystem` — использование диска
    - `procstat` — /proc/stat
  
- **`backend/src/routes/metrics-config.js`** — API для управления метриками
  - `GET /api/metrics-config` — список всех метрик
  - `POST /api/metrics-config/collect` — сбор всех включённых метрик
  - `GET /api/metrics-config/:id` — сбор одной метрики
  - `POST /api/metrics-config/add` — добавление пользовательской метрики
  - `DELETE /api/metrics-config/:id` — удаление метрики
  - `PATCH /api/metrics-config/:id/toggle` — включение/выключение
  - `PATCH /api/metrics-config/:id` — обновление метрики

- **`backend/src/routes/metrics.js`** — обновлён для включения пользовательских метрик в `/api/metrics`

| `/api/metrics-config/collect` | POST | Сбор всех включённых метрик |

### Пример пользовательской метрики
```json
{
  "id": "cpu-fan",
  "nameRu": "Вентилятор CPU",
  "name": "CPU Fan Speed",
  "type": "gauge",
  "unit": "RPM",
  "max": 8000,
  "color": "#10b981",
  "collect": {
    "method": "command",
    "command": "cat /sys/class/hwmon/hwmon0/fan1_input 2>/dev/null || echo 0",
    "regex": null,
    "interval": 5000
  },
  "description": "CPU fan speed from hwmon",
  "custom": true
}
```

### Шаблоны метрик
- 💨 Вентилятор CPU — `cat /sys/class/hwmon/hwmon0/fan1_input`
- 💨 Вентилятор GPU — `nvidia-smi --query-gpu=fans.speed --format=csv,noheader`
- 🌡️ Температура RAM — `sensors | grep -m1 "Temp" | grep -o "[0-9]\\+\\.?[0-9]*" | head -1`
- 💾 Температура NVMe — `smartctl -A /dev/nvme0n1 | grep Temperature | awk '{print $10}'`
