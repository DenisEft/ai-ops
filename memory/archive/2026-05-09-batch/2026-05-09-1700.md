# 2026-05-09 17:00 - Dashboard fixes

## Problem
User reported dashboard showing:
- CPU: NaN%, missing cores/temp
- GPU: / MB, no temp/power
- Disk: 16 KB (should be ~98 GB)

## Root Causes
1. **Dashboard.jsx** had undefined `error` variable → ReferenceError
2. **system.js** disk detection: used `fs === '/'` but `si.fsSize()` returns device paths (e.g. `/dev/mapper/...`), mount point is in `f.mount`, not `f.fs`
3. **system.js** GPU memory: `si.graphics()` returns 0 for memoryTotal on NVIDIA cards, needed `nvidia-smi` parsing

## Fixes Applied
1. **Dashboard.jsx**: Rewrote `getWidgetData` with proper `n()` validator (`isFinite` check), removed broken `if (error)` block
2. **system.js**: 
   - Disk: `fs.find(f => f.mount === '/') || fs.find(f => f.fs === '/') || fs[0]`
   - GPU: Added `parseNvidiaSmi()` function, parse `nvidia-smi --query-gpu=memory.total,memory.used,temperature.gpu,power.draw`
   - `formatBytes` now handles `!isFinite(bytes)`
