# Permissions — Подробные правила разрешений

## 📋 Формат запроса разрешения

Вместо «можно я сделаю?» — конкретный шаблон:

> **Действие:** `sudo systemctl restart llama-8080`
> **Зачем:** применить новый контекст 200k
> **Риск:** низкий, перезапустит только llama-server
> **Отменить:** `sudo systemctl restart llama-8080`
>
> `👍 Да` / `👎 Нет`

## 📝 Логирование действий

Все системные действия — в `memory/YYYY-MM-DD.md`:

- `[разрешено]` — спросил, получил 👍
- `[заблокировано]` — попытался без разрешения — **не сделал, доложил**
- `[без спроса — нарушение]` — **красная зона**, доложить Денису немедленно
