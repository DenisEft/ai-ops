# Coder Workspace

Рабочее пространство для субагента-кодера.

## Структура

```
coder/
├── AGENTS.md      # правила для субагента
└── README.md      # этот файл
```

## Как работает

1. Лора получает задачу от Дениса
2. Лора создаёт ТЗ для кодера
3. Кодер выполняет задачу в этой директории
4. Лора проверяет результат
5. Лора показывает Денису
